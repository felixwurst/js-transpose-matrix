# Transpose a Matrix in JavaScript

### Exercise:

Transpose the columns and rows of an existing array and insert them into a new array.

```
existing array = [
    [2,5,8,6,7,4,7],
    [5,9,2,3,6,6,8],
    [7,2,3,7,3,2,3],
    [5,1,4,2,8,1,2],
    [2,3,7,4,2,5,1]
]

new array = [
    [2,5,7,5,2],
    [5,9,2,1,3],
    [8,2,3,4,7],
    [6,3,7,2,4],
    [7,6,3,8,2],
    [4,6,2,1,5],
    [7,8,3,2,1]
]
```
